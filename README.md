# SQL E-commerce Sales Analysis

## Project Overview
This project analyzes e-commerce sales data using SQL to extract meaningful business insights related to revenue, customer behavior, and product performance.

## Tools Used
- MySQL

## Key Insights
- Calculated total revenue generated from sales
- Identified top customers based on spending
- Analyzed best-selling products
- Examined country-wise revenue distribution

## Project Files
- ecommerce_analysis.sql

## Skills Demonstrated
- SQL Joins
- Aggregation Functions (SUM, GROUP BY)
- Data Analysis
- Database Design
